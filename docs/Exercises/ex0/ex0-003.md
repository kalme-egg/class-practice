# 演習0-003 算数

## 問題文

小学生であるあなたは、算数の四則演算をならいました。
そして宿題が出されたのですが、いちいち筆算なんてやっていられません。
そこで、2つの数を入力すると四則演算を行ってくれるプログラムを作成することにしました。

$N$回の計算を行います。
$i$回目の計算では、演算の種類$C_i$と整数$A_i, B_i$が与えられるので、$A_i$と$B_i$を$C_i$で計算した結果を出力してください。
具体的には、

- $C_i$が`1`のとき
$A+B$を計算してください。

- $C_i$が`2`のとき
$A-B$を計算してください。

- $C_i$が`3`のとき
$A \times B$を計算してください。

- $C_i$が`4`のとき
$A \div B$を計算してください。ただし、小数点以下を切り捨てて整数で出力してください。


## 制約

- $1 \le N \le 100$
- $1 \le A \le 100$
- $1 \le B \le 100$
- $C$は$1, 2, 3, 4$のいずれか
- 入力は全て整数

## 入力

<div class="problem-code" markdown>

$N$ <br>
$C_1$ $A_1$ $B_1$ <br>
$C_2$ $A_2$ $B_2$ <br>
$\vdots$ <br>
$C_N$ $A_N$ $B_N$

</div>

## 出力

$N$行出力してください。
$i$行目では、$A_i$と$B_i$を演算した結果を出力してください。
