## Problem - 恐怖！人造人間の侵略

あなたは人造人間を作成しているマッドサイエンティストです。
あなたは人間クラスから人間インスタンスを作成する新しい技術を開発しました。

さて、技術を開発したら、試すべきでしょう！
人間の名前$S$と年齢$A$が入力されるので、それをもとに人間インスタンスを作成して、そのメンバ関数`displayInfo()`を実行するプログラムを作成してください。

### 与えられるファイル

??? info "Person.h"

    ```c++ linenums="1"
    #ifndef _PERSON_H_
    #define _PERSON_H_

    #include <string>

    class Person
    {
    private:
        std::string name;
        int age;

    public:
        Person();
        Person(std::string name, int age);

        void setName(std::string name) { this->name = name; }
        std::string getName(void ) { return name; }
        void setAge(int age) { this->age = age; }
        int getAge(void ) { return age; }

        void displayInfo(void );
    };

    #endif /* _PERSON_H_ */
    ```

??? info "Person.cpp"

    ```c++ linenums="1"
    #include "Person.h"
    #include <string>
    #include <iostream>

    using namespace std;

    Person::Person()
    {

    }

    Person::Person(string name, int age)
    {
        setName(name);
        setAge(age);
    }

    void Person::displayInfo(void )
    {
        cout << "名前 : " << getName() << endl;
        cout << "年齢 : " << getAge() << endl;
        return;
    }
    ```

### 制約

- $1 \le |S| \le 10$
- $0 \le A \le 10000$

### 入力
```
S A
```
