# 演習1-c01-1

(作問：zunda(kalme))

## 演習c0Xとは?

動画で扱ったクラスを、そのまま拡張する演習問題なのだ！

## Problem - ハッピーバースデーなのだ！

なぞのいきものに誕生日を与えるのだ。  
生まれた月を`birthMonth`、生まれた日を`birthDay`にするのだ。  
誕生日は...6月19日にするのだ！

### 与えられるファイル

??? info "Nazo.h"

	```c++ linenums="1"
	#ifndef _NAZO_H_
	#define _NAZO_H_

	class Nazo {
	public:
		float height;
		int age;
		bool alive;

		void sayHi(void );
	};

	#endif /* _NAZO_H_ */
	```

??? info "Nazo.cpp"

	```c++ linenums="1"
	// ここは変えなくて良さそうなのだ！
	#include "Nazo.h"
	#include <iostream>

	using namespace std;

	void Nazo::sayHi(void )
	{
		cout << "Hi!" << endl;
		return;
	}
	```

??? info "main.cpp"

	```c++ linenums="1"
	#include <iostream>
	#include "Nazo.h"

	using namespace std;

	int main(void ){
		Nazo foo;
		foo.height = 13.5;
		foo.age = 5;
		foo.alive = true;

		cout << "foo: " << endl;
		cout << "  height: " << foo.height << endl;
		cout << "  age: " << foo.age << endl;
		cout << "  birthday: " << foo.birthMonth << "/" << foo.birthDay << endl; // 6/19 って出したいのだ！
		return 0;
	}
	```

## 提出

コードを編集して、「提出」を押したらチェックされるのだ。
