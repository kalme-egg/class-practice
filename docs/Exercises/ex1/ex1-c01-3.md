## Problem - エラーフィックス、その1

中身のおかしいクラスを直してみヨウ。  
コンパイルエラーの中身と、各ファイルで間違っている行数をヒントにしてあるヨ。  

はじめのウチからエラーなしで書ききるのは難しいヨ。
エラーを直す力モ身に付けていコウ。  

### コンパイルエラーについて

環境によってエラーの出方が違うコトもあるヨ。  
Jenkinsトカ、いつものgccに近いコンパイルエラーの文をヒントにしてあるヨ。  
見慣れないエラーが出て戸惑ったときハ、ヒントのものも見てミテネ。  

### 与えられるファイル

??? info "Nazo.h"

	```linenums="1"
	#ifndef _NAZO_H_
	#define _NAZO_H_

	class Nazo {

		float height
		int age;
		bool alive;

		void sayHi(void );
	}

	#endif /* _NAZO_H_ */
	```

??? info "Nazo.cpp"

	```linenums="1"

	#include <iostream>

	using namespace std;

	void Nazo:sayHi(void )
	{
		cout << "Hi!" >> endl;
		return;
	}
	```

??? info "main.cpp"

	```linenums="1"
	#include <iostream>
	#include "Nazo.h"

	using namespace std;

	int main(void ){
		Nazo foo;
		foo height = 13.5;
		foo.age = 5;
		foo.alive = true;

		cout << "foo: " << endl;
		cout << "  height: " << foo.height << endl;
		cout << "  age: " << foo->age << endl;

		foo.sayHi();
		cout << endl;

		return 0;
	}
	```
